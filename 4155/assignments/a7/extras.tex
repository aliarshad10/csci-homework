Code from Kalman Filter. In higher dimensions mutliplication would need to be matrix dot product. Constants set as parametres for easier testing.
\begin{verbatim}
def kf_predict(X, S, U):
	x = A*X + B*U
	s = A*S*A + R
	return x, s

def kf_update(X, S, Z):
	k = S*C/(C*S*C+Q)
	x = X+k*(Z-C*X)
	s = (1-C*k)*S
	return x, s
\end{verbatim}
